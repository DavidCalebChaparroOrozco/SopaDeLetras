Palabras

Glosario:

HTML

Es un lenguaje de marcado que se utiliza para desarrollar paginas de Internet. Sus siglas corresponden a HyperText Markup Language, es decir Lenguaje de Marcas de Hipertexto. Tiene como objetivo estructurar y desplegar una página web y sus contenidos

CSS

Es un lenguaje de hojas de estilos creado para controlar la parte estética o el aspecto de los documentos definidos con HTML. Es conocida porque permite separar los contenidos, presentación ya es imprescindible para crear páginas webs complejas.

JavaScript

Es un lenguaje de programación que permite implementar funciones mas complejas en páginas web, es utilizada para actualizaciones de contenidos, mapas interactivos, animaciones, entre otras cosas.

Visual Studio Code

Es un editor de código fuente desarrollado por Microsoft, es un editor de texto muy popular porque incluye soporte para depuración, control, resaltado de sintaxis, permite añadir nuevas extensiones que ayudan a los desarrolladores a trabajar de manera mas eficiente, es conocido por ser personalizable, gratuito y de código abierto.

PHP

Es un lenguaje de programación con el fin del desarrollo de aplicaciones web, favoreciendo la conexión entre los servidores y la interfaz de usuario. Es popular porque es de código abierto y tiene constantes actualizaciones.

Github

Es una plataforma creada para alojar códigos de las aplicaciones de cualquier desarrollador. Su objetivo principal es alojar el código de las aplicaciones y herramientas del desarrollador, es conocida porque permite que demás desarrolladores puedan leer sobre su desarrollo y colaborar.

DOM

Es una interfaz de programación para los documentos HTML, que permite una facilidad a la hora de representar una estructura donde se define de que manera los programas accede, modifican tanto su estructura, estilo y contenido.

Angular

Es un framework que fue desarrollado por Google para facilitar la creación y programación de sitios web de una sola página. Es conocido porque separa el frontend y el backend de la aplicación, con el fin de evitar escribir código repetitivo y mantiene todo mas ordenado gracias al patrón MVC (Modelo-Vista-Controlador)

React

Es una librería de Javascript con enfoque en el desarrollo de interfaces de usuario, es muy utilizado en porque cubre con un ecosistema muy amplio ya sean de módulos, herramientas y componentes capaces de cubrir objetivos con poco esfuerzo. Es una ayuda porque facilita el desarrollo porque ofrece cosas ya listas, en las que los desarrolladores no necesitan invertir tiempo de trabajo.

Vue

Es un framework para construir interfaces de usuario. Su librería esta centrada en la capa de visualización, y es fácil de utilizar e integrar con otras librerías o proyectos ya realizados.

Redux

Es un patrón de arquitectura de datos que permite manejar los estados de una aplicación de forma predecible, es utilizado para reducir el numero de relaciones entre componentes de la aplicación y mantener un flujo de datos sencillo.

NodeJS

Es un entorno de ejecución para Javascript que permite ejecutar en el servidor de forma asíncrona, tiene una arquitectura orienta a eventos, es conocida por ser una plataforma muy presente en el mercado y que se encuentra en constantes avances.

SASS

Es un preprocesador de CSS, es decir es una herramienta que permite generar, de forma automática, hojas de estilo, con características que no tiene CSS, y que es propia de los lenguajes de programación, como son variables, funciones, herencias y entre otras cosas.

TypeScript

Es un lenguaje de programación, que requiere tipos de datos de las variables, los tipos de datos son estrictos y son respetados, se suele implementar en FrontEnd o BackEnd, brinda control sobre las aplicaciones ante de correr el código.

Drupal

Es un Sistema de gestión de contenidos (CMS), que es conocido por su potencia y flexibilidad, que es muy utilizada para los usuarios que desarrollan una página web dinámica con diversas funciones. Es un gestor de contenidos con diversos tipos de sitios como pueden ser blogs hasta grandes páginas de Internet.

GraphQL

Es un lenguaje de consultas, que permite hacer consultas y esperar una respuesta que es predecible, SQL no tiene ninguna relación con GraphQL porque la función de GraphQL es para comunicar cliente y servidores, es una herramienta muy útil que propone la optimización y traslación de información del servidor al cliente.

WordPress

Es un Sistema de gestión de contenidos (CMS), que permite crear y mantener diversos tipos de web, es un sistema sencillo e intuitivo para realizar todo tipo de web complejas, adicional es un sistema ideal para un sitio web se actualice periódicamente.

Bootstrap

Es un framework, que permite crear interfaces de usuario, con una adaptabilidad muy alta que ha ido evolucionando es muy popular entre desarrolladores.

Responsive

Es un termino para aquellas paginas web que son capaz de adaptarse a cualquier dispositivo donde se visualice. Es de tener en cuenta ya que Google penaliza en su buscador a aquellas paginas que no se encuentren Responsive.

URL

Significa Uniform Resource Locator, es una dirección que es dada a un recurso en la Web, cada url valida apunta a un único recurso.
